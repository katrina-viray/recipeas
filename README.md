### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/katrina-viray/recipeas.git
   ```
2. Go into folder
   ```sh
   cd ./recipeas
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run website locally
   ```sh
   pnpm run dev
   ```
